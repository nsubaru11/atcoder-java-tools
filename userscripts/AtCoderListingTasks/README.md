# AtCoder Listing Tasks

AtCoder のコンテストページにおいて、「問題」タブから各問題へ素早くアクセスするためのクイックナビゲーション UserScript です。

## 概要

通常、特定の問題を開くには「問題」タブをクリックして一覧に移動する必要がありますが、このスクリプトを導入するとタブにマウスを置くだけで問題リストがドロップダウン表示されます。コンテスト中の問題切り替えを最小限のクリック数で実現します。

## 主な機能

- **ホバーアクセス**: 「問題」タブにカーソルを合わせるだけで、Aから順に問題タイトルがリストアップされます。
- **ダイレクト遷移**: リスト内の各問題をクリックすることで、現在のページから直接その問題文へ遷移できます。
- **コンテスト情報表示**: 問題タイトルだけでなく、配点などの付随情報を表示することも可能です。
