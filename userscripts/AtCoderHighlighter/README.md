# AtCoder Highlighter

AtCoder の問題文に含まれる重要な情報（制約、変数、制限時間など）を強調表示し、読解ミスを防ぐための視認性向上 UserScript です。

## 概要

問題文の中で見落としやすい「実行時間制限」や「メモリ制限」、また数式（KaTeX）として記述された変数や制約の数値を自動で判別してハイライトします。複雑な問題文でも重要なパラメータを素早く把握できるようになります。

## 主な機能

- **制約・変数ハイライト**: `var` タグや数式内の数値を強調し、条件の見落としを減らします。
- **制限時間の強調**: ページ上部の実行時間制限やメモリ制限を際立たせ、計算量の検討をサポートします。
- **非破壊的な装飾**: ページのレイアウトを崩さず、標準のスタイルに馴染む形で強調を行います。
