# AtCoder Performance Overlay

AtCoder のユーザーページにあるレーティンググラフ上に、パフォーマンス推移のグラフを重ねて表示し、実力の変化をより詳細に可視化する
UserScript です。

## 概要

AtCoder
標準のグラフはレーティング（蓄積値）のみを表示しますが、このスクリプトを導入することで各コンテストでの「パフォーマンス（その時の実力値）」を同時に確認できるようになります。レーティングの増減だけでなく、どれだけの成績を収めたかを直感的に把握できます。

## 主な機能

- **オーバーレイ表示**: レーティンググラフと同じ座標軸上にパフォーマンスの折れ線グラフを重ねて描画します。
- **表示切り替え**: 専用の ON/OFF ボタンにより、パフォーマンスグラフの表示・非表示をいつでも切り替え可能です。
- **期間絞り込み（all/recent）**: 全期間の表示と、直近の数回分に絞った表示をボタン一つで切り替えられます。
- **ステータス連動**: グラフ上の点にマウスホバーすると、その時のレート、パフォーマンス、順位などの詳細情報を同期して表示します。

## インストール
